# Kombuchator

Staticka ceska webova aplikace pro orientacni vypocet nalevu na F1 kombuchu.

## Spusteni

Otevri `index.html` v prohlizeci, nebo spust lokalni staticky server:

```bash
python3 -m http.server 4173
```

## Scope MVP

- Klasicka kalkulacka
- Hokus pokus
- Realtime prepocet bez tlacitka
- Zakladni validace a varovani
- Edukacni modal `Jak to funguje?`
